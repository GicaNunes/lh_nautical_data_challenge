# 🚢 LH Nautical Data Challenge

Este repositório reúne minha jornada no **LH Nautical Data Challenge**, onde explorei dados reais de vendas, clientes e produtos para construir análises e um dashboard interativo em **Streamlit**.

---

## 🌟 Sobre o Projeto
O desafio consistiu em:
- Limpar e padronizar dados com inconsistências (categorias duplicadas, erros de digitação, acentos).
- Responder questões analíticas (estatísticas, lucro por cliente, distribuição de categorias).
- Criar visualizações interativas para facilitar a tomada de decisão.
- Publicar o dashboard online para acesso público.

## 🔑 Principais Aprendizados
- **ETL de dados**: tratamento de CSV e JSON, normalização de categorias e remoção de duplicidades.
- **Visualização**: uso de Plotly e Streamlit para gráficos dinâmicos e interativos.
- **Deploy**: configuração de `requirements.txt`, ajustes de ambiente e publicação no Render.
- **Boas práticas**: padronização de código, limpeza de repositório e documentação clara.

---

## 📊 Destaques do Dashboard
- **Questão 2**: Estatísticas gerais de vendas.  
- **Questão 3**: Lucro acumulado por cliente.  
- **Questão 8**: Distribuição de categorias (corrigida e padronizada).  
- **Explorações adicionais**: vendas mensais, semanais, top produtos e pares de produtos comprados juntos.  

---

## 💡 Reflexões Finais
Essa jornada mostrou como **dados brutos podem se transformar em insights valiosos** quando tratados com cuidado e apresentados de forma visual.  
O dashboard é uma ferramenta prática para explorar os resultados e apoiar decisões estratégicas.
